# BBXAI

**BBXAI** / **BB × AI** is an AI-assisted loopstation concept for beatboxers and vocal sound-makers.

The product goal is simple:

> Record human-made beatbox sounds. Let the system arrange, clean, balance, and structure them into music.

BBXAI is not meant to be a prompt-based AI music generator. It is an arranger and production assistant for sounds created by the user.
