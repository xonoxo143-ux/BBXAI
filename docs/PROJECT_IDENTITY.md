# Project Identity

## Name

**BBXAI** is the repository and product name.

**BB × AI** is the visual/brand reading.

Meaning:

> Beatbox × Artificial Intelligence

## Core idea

BBXAI helps people turn human-made beatbox sounds into arranged music.

The user creates the sounds. The system assists with arrangement, timing, balance, and production polish.

## What BBXAI is

BBXAI is:

- an AI-assisted beatbox loopstation concept
- a song-arrangement helper for vocal sounds
- a production bridge for amateurs
- a tool for people with raw sound creativity but limited production skill

## What BBXAI is not

BBXAI is not:

- a generic AI music generator
- a replacement for beatboxers
- a prompt-to-song machine
- a tool that hides the human performance
- a DAW clone

## Product sentence

> BBXAI turns human-made beatbox sounds into arranged music without making the user feel replaced.

## Current design stance

The project should protect the feeling that the user made the track.

The AI should mainly answer:

- where should this sound go?
- how should the section build?
- what should be louder or cleaner?
- when should layers enter or leave?
- how can this become a track instead of loose loops?
